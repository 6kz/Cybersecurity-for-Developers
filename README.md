# 🛡️ Cibersegurança para Iniciantes: O Teu Primeiro Passo

Olá! Este repositório foi criado pelo **Tomás Matos** como material de apoio para a palestra **"Navegar na Cibersegurança"**. 

Se estás a começar agora no mundo da programação, este é o teu guia prático para escreveres código que não se deixa "partir".

---

## 🚀 O que vais encontrar aqui?

Neste repositório, vais aprender a identificar e corrigir as 3 falhas mais comuns que afetam quem está a começar:

1.  **SQL Injection:** Como impedir que alguém "fale" diretamente com a tua base de dados.
2.  **XSS (Cross-Site Scripting):** Como garantir que o teu site não executa código malicioso.
3.  **Broken Access Control:** Como ter a certeza de que um utilizador só vê o que lhe pertence.

## 💡 Dicas de Ouro para Iniciantes

> **Regra nº 1: Nunca confies no utilizador.** > Tudo o que alguém escreve no teu site deve ser tratado como perigoso até ser "limpo" pelo teu código.

### Ferramentas Úteis:
* **SonarLint:** Uma extensão para o VS Code que te avisa de erros enquanto escreves.
* **Snyk:** Verifica se as bibliotecas que instalaste têm vírus ou falhas conhecidas.
* **OWASP Top 10:** A lista oficial das coisas que mais correm mal na internet.
